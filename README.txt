# Компонент комментариев на Web Components

Проект представляет собой реализацию системы комментариев с использованием современных веб-технологий, в частности Web Components. Компонент полностью инкапсулирован, переиспользуем и может быть добавлен на любую страницу.

## Функциональные возможности

- Отображение, добавление, редактирование и лайки комментариев
- Вложенность комментариев (ответы на комментарии)
- Сортировка комментариев (новые, старые, популярные)
- Поддержка прикрепления изображений и GIF файлов
- Модальное окно для просмотра изображений
- Темная и светлая темы интерфейса
- Локальное хранение данных (localStorage)
- Адаптивный дизайн

## Технологии

- **Web Components**: создание собственных HTML-элементов
- **Shadow DOM**: инкапсуляция стилей и структуры
- **HTML Templates**: использование шаблонов для удобного переиспользования
- **Custom Elements API**: определение собственных элементов
- **CSS Custom Properties**: гибкая стилизация и темизация
- **Slots**: механизм для композиции компонентов и вложенности комментариев
